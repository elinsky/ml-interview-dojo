# Parametric vs Nonparametric Tests

**Q:** What's the difference between parametric and nonparametric tests?
- Key distinction
- When to use nonparametric
- Examples of each
- Tradeoff

**A:**

- **Parametric**: assumes data follows specific distribution (e.g., normal); estimates parameters
- **Nonparametric**: makes no distributional assumptions; often uses ranks
- **Use nonparametric when**:
  - Data is ordinal
  - Distribution is unknown or clearly non-normal
  - Small sample size
  - Outliers present
- **Examples**: t-test (parametric) vs Mann-Whitney U (nonparametric); Pearson (parametric) vs Spearman (nonparametric)
- **Tradeoff**: nonparametric is more robust but less powerful when parametric assumptions hold
