# What is a Decision Tree

**Q:** What is a decision tree?

**A:**

- **Binary tree** for classification or regression (aka **CART** — Classification And Regression Trees)
- Each **internal node** = input variable + split point
- Each **leaf** = prediction (class label or value)
- Partitions input space into **rectangles/hyper-rectangles**
- **Inference**: O(log₂ m) — traverse tree depth
- Foundation for Random Forest, Gradient Boosting, XGBoost
