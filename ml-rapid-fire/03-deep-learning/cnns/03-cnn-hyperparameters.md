# CNN Hyperparameters

**Q:** What are the key hyperparameters in a convolutional layer?

**A:**

- **Filter size (F)**: typically 3x3 or 5x5
- **Number of filters (K)**: determines output depth
- **Stride (S)**: step size when sliding (usually 1)
- **Padding (P)**: "same" preserves size, "valid" shrinks
- Common setting: F=3, S=1, P=1
