# When to Choose L1 (Lasso)

**Q:** When would you choose L1 regularization (Lasso)? (4 reasons)

**A:**

- For **feature selection** (forces some weights = 0)
- When many features are **irrelevant or redundant**
- For a **sparse, interpretable** model
- When you can tolerate **instability with correlated** predictors
