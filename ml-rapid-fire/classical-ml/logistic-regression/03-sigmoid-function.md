# Sigmoid Function

**Q:** What is the sigmoid function and its formula?

**A:** A function that maps any real number to a value between 0 and 1.

$$f(x) = \frac{1}{1 + {\mathrm e}^{-x}} = \frac{{\mathrm e}^{x}}{{\mathrm e}^{x} + 1}$$
