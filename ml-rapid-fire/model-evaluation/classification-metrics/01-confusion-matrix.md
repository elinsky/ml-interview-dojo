# Confusion Matrix

**Q:** What is a confusion matrix?

**A:**

- 2x2 table for binary classification: TP, TN, FP, FN
- Rows = actual class, columns = predicted (convention varies)
- **TP**: correctly predicted positive
- **FP**: predicted positive, actually negative (Type I error)
- **FN**: predicted negative, actually positive (Type II error)

**See also:** [[202101072002 Bayes Theorem]] (for Bayesian connection to metrics)
