# F1 Score

**Q:** What is F1 score and when should you use it?

**A:**

- F1 = 2 * (Precision * Recall) / (Precision + Recall)
- **Harmonic mean** of precision and recall
- Punishes extreme values (low P or R tanks F1)
- Use when you need balance between P and R
- Good single-number summary for imbalanced datasets
