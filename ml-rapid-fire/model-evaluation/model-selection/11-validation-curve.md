# Validation Curve

**Q:** What is a validation curve?
- What is on the x and y axes

**A:**

- Plot of training and validation performance (y-axis) vs hyperparameter value (x-axis)
  - Y-axis can be score (higher = better) or error (lower = better)
- Shows effect of a single hyperparameter on model performance
