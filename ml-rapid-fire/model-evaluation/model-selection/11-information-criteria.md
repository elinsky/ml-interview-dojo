# Information Criteria (AIC/BIC)

**Q:** What are AIC and BIC for model selection?

**A:**

- AIC/BIC: penalize model complexity + training error
- No separate validation set needed
- BIC penalizes complexity more heavily than AIC
- Lower is better; compare models on same data
