# Validation Set Purpose

**Q:** What is a validation set and how does it differ from test set?

**A:**

- Validation: tune hyperparameters and model selection
- Test: final unbiased evaluation (used once)
- Validation can be used repeatedly during development
- Common split: 60/20/20 or 70/15/15
