# Holdout vs Cross-Validation

**Q:** When should you use simple holdout vs cross-validation?

**A:**

- Holdout: large datasets, expensive models
- Cross-validation: smaller datasets, need reliable estimates
- CV reduces variance in performance estimate
- Holdout is faster but wastes data
