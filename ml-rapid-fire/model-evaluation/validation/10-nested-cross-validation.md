# Nested Cross-Validation

**Q:** What is nested cross-validation and when is it needed?

**A:**

- Outer loop: estimate generalization performance
- Inner loop: hyperparameter tuning
- Prevents optimistic bias from tuning on same data
- Use when both tuning and evaluating performance
