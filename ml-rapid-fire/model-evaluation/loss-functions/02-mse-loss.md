# MSE Loss

**Q:** What is Mean Squared Error loss and when do you use it?

**A:**

- MSE = (1/n) Σ(y - ŷ)²
- Standard loss for regression problems
- Penalizes large errors more heavily (quadratic)
- Sensitive to outliers
