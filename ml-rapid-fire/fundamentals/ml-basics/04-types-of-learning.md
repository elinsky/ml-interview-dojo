# Types of Machine Learning

**Q:** What are the four main types of machine learning?

**A:**
- **Supervised**: Training data includes desired outputs (labels)
- **Unsupervised**: Training data has no labels (e.g., clustering)
- **Semi-supervised**: Only some training data has labels
- **Reinforcement**: Agent learns from rewards/penalties over sequences of actions
