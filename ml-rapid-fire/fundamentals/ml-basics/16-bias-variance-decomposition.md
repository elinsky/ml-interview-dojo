# Bias-Variance Decomposition

**Q:** What is the bias-variance decomposition of error?

**A:**

- **Formula**: E[(y₀ - f̂(x₀))²] = Var(f̂) + Bias(f̂)² + Var(ε)
- **Variance**: How much f̂ changes with different training data
- **Bias²**: Error from model being an oversimplification of true f
- **Irreducible error (ε)**: Cannot be reduced given only X; requires additional predictors

[@james2013introduction, p. 33-35]
