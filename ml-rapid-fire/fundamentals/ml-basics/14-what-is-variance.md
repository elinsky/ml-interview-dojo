# What is Variance (Bias-Variance)?

**Q:** What is variance in the context of bias-variance?

**A:**
- The amount by which f̂ would **change if estimated using a different training dataset**
- A property of the learning method (e.g., decision trees tend to have high variance)

"Variance refers to the amount by which f̂ would change if we estimated it using a different training data set." [@james2013introduction, p. 34]
