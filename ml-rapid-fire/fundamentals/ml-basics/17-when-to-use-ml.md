# When to Use Machine Learning

**Q:** When is machine learning a good approach? (Name 4 scenarios)

**A:**

- **No human expert exists** (true discovery problems)
- **Humans can do it but can't explain how** (e.g., recognizing faces, driving)
- **Problem changes frequently** (e.g., stock market, spam detection)
- **Each user needs customization** (e.g., recommendations)
