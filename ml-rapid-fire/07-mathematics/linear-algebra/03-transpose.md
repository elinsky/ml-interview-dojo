# Matrix Transpose

**Q:** What is a matrix transpose and what are its key properties?
- Definition
- Key properties
- Special matrices related to transpose

**A:**

- **Definition**: (Aᵀ)ᵢⱼ = Aⱼᵢ — flip rows and columns
- **Properties**:
  - (Aᵀ)ᵀ = A
  - (A + B)ᵀ = Aᵀ + Bᵀ
  - (AB)ᵀ = BᵀAᵀ (reverse order!)
  - (cA)ᵀ = cAᵀ
- **Special matrices**:
  - Symmetric: A = Aᵀ (covariance matrices)
  - Orthogonal: AᵀA = AAᵀ = I (rotation matrices)
