# Mutually Exclusive vs Independent

**Q:** What is the difference between mutually exclusive and independent events?

**A:**

- Mutually exclusive: cannot both occur, P(A ∩ B) = 0
- Independent: occurrence of one doesn't affect other
- Mutually exclusive events are NOT independent
- If A occurs, B cannot (P(B|A) = 0 ≠ P(B))
