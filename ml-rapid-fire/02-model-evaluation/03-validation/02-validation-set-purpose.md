# Validation Set Purpose

**Q:** What is a validation set and how does it differ from test set?
- Validation set purpose
- Test set purpose
- Key difference in usage

**A:**

- **Validation**: tune hyperparameters and model selection (can use repeatedly)
- **Test**: final unbiased evaluation (used once)
- **Key difference**: validation used during development, test reserved for final evaluation
