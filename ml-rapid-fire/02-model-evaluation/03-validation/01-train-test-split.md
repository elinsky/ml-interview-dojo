# Train/Test Split

**Q:** Why do we split data into train and test sets?
- Train set purpose
- Test set purpose
- Key rule about test set usage

**A:**

- **Train set**: fit model parameters
- **Test set**: evaluate generalization to unseen data
- **Key rule**: test set should only be used ONCE at final evaluation (prevents overfitting to test data)
